ND2 Viewer

Version 1.1.0 (January 2025)

The ND2 viewer is a simple visualizer of ND2 files produced by NIS Elements software. The program is under development with more features and improvements coming soon.

Capabilities

  • Visualize ND2 files in both 2D image view and 3D volume view.
  • View individual channel or all channels.
  • View binary layers.
  • Use look-up-tables (LUTs) to enhance image contrast for viewing.
  • Navigate through the ND2 frames in all dimensions.
  • Explore image attributes.

Download

  • Nd2Viewer.exe single file executable for Windows x64
  • Nd2Viewer_MacOS_aarm64.zip compressed bundle for macOS (Apple Silicon)
  • Nd2Viewer_MacOS_x64.zip compressed bundle for macOS (Intel x86_64)
